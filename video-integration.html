🎥 视频集成指南

本指南将帮助您将采访视频集成到客家文化网站中。

1 准备视频文件

确保您的视频文件为MP4格式，建议分辨率不超过1920x1080，文件大小控制在100MB以内。

2 创建视频目录

在项目根目录下创建 videos 文件夹，并将视频文件放入其中。

3 更新HTML结构

在 index.html 的采访部分，将视频占位符替换为实际的视频元素：

<div class="video-section"> <h3>采访视频一</h3> <div class="video-container"> <video controls> <source src="videos/interview1.mp4" type="video/mp4"> 您的浏览器不支持视频播放。 </video> </div> </div>
4 更新JavaScript

在 script.js 中，修改视频模态框的源地址：

// 在 showVideoModal 函数中更新视频源 <video controls> <source src="videos/interview1.mp4" type="video/mp4"> 您的浏览器不支持视频播放。 </video>

📁 推荐的文件结构

hakka-culture-website/
├── index.html
├── styles.css
├── script.js
├── videos/
│   ├── interview1.mp4    # 客家文化传承人访谈
│   ├── interview2.mp4    # 客家建筑专家访谈
│   └── background.mp4    # 背景视频（可选）
└── README.md

🎬 视频格式建议

  • 格式：MP4 (H.264编码)
  • 分辨率：1920x1080 或 1280x720
  • 帧率：25fps 或 30fps
  • 比特率：2-5 Mbps
  • 音频：AAC编码，128kbps

🔧 高级配置

如果您需要更高级的视频功能，可以考虑以下选项：

1. 自动播放（静音）

<video autoplay muted loop> <source src="videos/background.mp4" type="video/mp4"> </video>

2. 视频预览图

<video controls poster="images/video-thumbnail.jpg"> <source src="videos/interview1.mp4" type="video/mp4"> </video>

3. 响应式视频

.video-container { position: relative; width: 100%; height: 0; padding-bottom: 56.25%; /* 16:9 宽高比 */ } .video-container video { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }

⚠️ 注意事项

  • 确保视频文件有适当的版权许可
  • 考虑视频文件大小对网站加载速度的影响
  • 提供视频的替代文本或描述
  • 测试在不同设备和浏览器上的兼容性
  • 考虑添加字幕或文字记录

🚀 性能优化建议

  • 使用 懒加载 延迟加载视频
  • 提供多种分辨率的视频文件
  • 使用 WebM 格式作为MP4的备选
  • 压缩视频文件以减少带宽使用
  • 考虑使用CDN托管视频文件

完成以上步骤后，您的客家文化网站就可以播放真实的采访视频了！
